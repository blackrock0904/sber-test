Необходимо:
 
1.	Использовать GIT для хранения кода (github или bitbucket)
2.	Cверстать представленный макет screen.png. При верстке можете использовать любой web-шрифт.
3.	При помощи React JS реализовать:
3.1	Загрузку данных  из data.json и их отображение в таблице.
3.2	При клике на заголовок столбца осуществлять сортировку таблицы по данному столбцу.
3.3	 Отображать цветовые индикаторы :
3.3.1	Длина закрашенного участка индикатора пропорциональна отношению максимального модуля значения к модулю значения в данной строке. 
3.3.2	Цвет окраски и направление определяется знаком значения в строке. 
3.3.3	fa.fa_data.r – данные таблицы, fa.fa_data.axis.r – заголовки столбцов (столбцов может быть произвольное количество), fDeltaPlan – значение для отображения цветового индикатора.
4.	По результатам выполнения тестового задания необходимо предоставить ссылку на открытый GIT репозиторий (github или bitbucket) который содержит код проекта.
